Chrome有用插件的收集

发表于 2017-07-01

好的Chrome插件能帮助我们更好的工作，提高效率

下面是我使用过感觉好的Chrome的插件

Postman

调试接口使用的利器

可以发送到桌面为快捷方式，方面快速启动

启动之后的界面为

基本上各种方式的请求都可以满足

使用方法非常简单，功能非常强大

JSON Viewer

请求的数据自动格式化
在浏览器上格式json格式的数据的,方便查看请求返回的json数据

比如请求豆瓣的电影api http://api.douban.com/v2/movie/subject/1764796
可以伸缩的返回结果为：

掘金

每天打开浏览器，查看最新的掘金技术文章

安装成功之后，打开浏览器就会这样的效果的

Block

该插件名称为 adblock
屏蔽网站广告的，防止影响阅读

效果设置

若把标题写为 adBlock 生成的h4标签会自带一个id=adblock的属性，会导致标题隐藏不显出来

Sourcegraph for GitHub

这个非常强大，可以从github跳转到Sourcegraph在线预览代码,进行其他操作，相当于一个强大在线的IDE

比如:

跳转到：

新媒体管家

方便对公众号的管理，增强公众号富文本编辑器的功能

安装之后

octotree

Code tree for GitHub,方便查看github仓库文件树

安装成功之后：

gitcodetree

该插件和 octotree 的功能一样，是基于octotree，不过这是国内开发的.
可以用在 码云  和 github上,而 octotree 只能用在 github上

有道词典Chrome划词插件

划词翻译，选中一个词语就会自动翻译

安装成功之后的效果：

视频小窗播放插件 PIP video


安装这个插件要按照提示，开启Chrome的实验特性chrome://flags/#enable-panels

安装后，打开视频会左上角有个图标

然后点击图标就可以小窗播放了。效果图：

Project Naptha

处理网页中的图片中的文章，你可以突出，清除，以及复制和粘贴，甚至编辑和翻译以前被困在图像中的文字

打开网页后，会自动监听打开页面的图片。将鼠标放到图片的文字区域，如果识别了则鼠标变成一条小竖线，然后点击右键会出现功能列表，如下图：

然后就可以处理图片中的文字了！

DevDocs

能快速查看api文档

安装完成之后自动跳转到该页面

这个右键点击显示出来，没有添加快捷方式的功能，就难受了。每次都得进到上面那个页面

GitHub Hovercard

GitHub Hovercard provides neat hovercards for GitHub. It brings you instant access to user/repo/issue/commit information from anywhere on GitHub

安装完成后，你可以在Github的任何位置，访问用户／仓库／issue/commit的信息

Axure RP Extension for Chrome

An extension that allows local viewing of Axure RP prototypes from Google Chrome

允许在chrome上查看Axure原型的插件

安装完成，需要早这里勾选上，允许访问本地文件

这样就可以查看原型文件了。

Set Character Encoding

打开一个网页后,右键右键可以设置网页编码

Termius - SSH Client

ssh 客户端,是一个应用,官方网站
插件地址 还可以使用对应系统的客户端

NIM

利用 Chrome Devtools 的 node 调试工具说明文档

Web Scraper

爬取网页的数据的利器,可以导出 csv格式的数据

使用的时候打开 dev toos

dualsub-supports

YouTube 播放器实现双语字幕 , 但是这个插件只能开发者模式安装 github 地址为 DualSub

安装之后的效果为:

WEB前端助手(FeHelper)

各种前端小工具的集合

安装之后的效果为:

Chromoji

Emoji for Google Chrome 在 chrome 使用 emoji 表情

安装之后的效果为:

DevTodo

在 Chrome 开发者工具上管理todo list

安装之后的效果为:

Console Importer

在浏览器控制台 加载 JS 或者 CSS 资源 方便在浏览器中使用对应加载的工具包


有的网站有安全策略 CSP 对资源加载限制， 比如 Github 这个插件是不可以使用的

Smallpdf

一个免费的 PDF 处理的工具机

XPath Helper

利用XPath 语法在页面长查询对应的元素

Office Editing for Docs, Sheets & Slides

Chrome在线查看和简单修改Office文件，包括Word, Excel,PPT,只需要把文件拖到浏览器中即可

The Fucking Github

管理Github已经Star的项目
View starred repositories, organizing stars, searching stars and searching repositories online for Github。

Refined GitHub

Refined GitHub 是一个简化 GitHub 界面并拥有实用功能的浏览器扩展程序

draw.io Diagrams

draw.io Diagrams 是一个画各种图的浏览器app应用

HTTP/2 and SPDY indicator

检测网站是否支持 HTTP/2 或者 SPDY 协议

Git History Browser Extension

在Github网站上查看文件的Git提交历史记录

SimilarWeb - 网站流量来源和排名

可以查看网站排名的具体情况

网页截图:注释&录屏

录屏，捕获整个页面或任何部分(还可以进行滚动长截图)

二维码生成器

可以方便地把当前页面转化成二维码，也可以把网页上任何文本或链接，甚至是您输入的任意内容都转化成二维码

以上插件下载的安装的时候，或需要翻墙进行科学上网
以上只是写出了，每个插件的用途的，具体的使用方法需自己去摸索

学到手上的才是自己的


扫描关注，查看更多文章，提高编程能力
